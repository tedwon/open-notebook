== How to extract tar.gz?
[source,bash,options="nowrap"]
----
$ tar -zxvf <FILE>.tar.gz
----


== How to extract tar.gz to target directory?
[source,bash,options="nowrap"]
----
$ tar -zxvf <FILE>.tar.gz -C <TARGET_DIR>
----
